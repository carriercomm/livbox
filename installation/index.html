Install on Linux and Mac

  1. Check if you have a cURL installed on your system.

  2. Download from .zip

    Download LivBox

  3. or Download From GitHub

    git clone https://github.com/enytc/livbox.git

  4. Configure Database

    cd livbox

    Open file app/config/database.php and edit database options: hostname, user and password.

  5. Create a VHost in /path/to/apache/sites-available/. In this example we'll use Apache.

    <VirtualHost *:80>
  ServerAdmin webmaster@localhost
  ServerName livbox.dev
  ServerAlias www.livbox.dev

  DocumentRoot /path/to/livbox/public

  <Directory /path/to/livbox/public>
    Options Indexes FollowSymLinks MultiViews
    AllowOverride All
    Order allow,deny
    Allow from all
  </Directory>
</VirtualHost>

  6. Run LivBox Installer

    php livbox install
