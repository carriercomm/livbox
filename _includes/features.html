File Manager

You can use the main features for managing files and folders, such as Download, Rename, Delete, and Share Files.

Share Files

You can share files with other users of the system or who has access to your shared link.

User Management

You manage your users, add, edit and remove and restart your user session, can also see access information like last login, last access and if the user is online.

Settings

You can edit the settings of the system as system title, description, language and time zone.

Login System

You can log into the system or recover your password if you missed it.

Session Security

The LivBox has a security system per session, you can not use the system in different places at the same time. The system has a safety lock so it makes it catches your login session and unlocks only when you logout. This will give more security to your data and changes. If you forget to log out, your session will be locked. After 30 minutes of no activity it will be unlocked.

Public Downloads

Once you share a file yours, you can pass the shared link to others. It can download to your computer or send it to your personal folder.

Protected Pages

The LivBox has a system allowing access to certain pages of the system. Only certain types of users can access important system pages.

Maintenance Mode

The LivBox has a maintenance mode, you turn on and off easily if needed.

Mobile Version

You can use all features of LivBox via computers, laptops, tablets or smartphones.
